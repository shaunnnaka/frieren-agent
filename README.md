<details>
<summary>フリーレンスコアについて</summary>

本当はスコアなんてつけたくないよ。でも人間はこういうのがないと安心しないらしいから。

| 軸 | 何を見ているか |
|----|--------------|
| **「なぜ作った」度** | 存在理由が謎なほどいい。「くだらないね」は最高の褒め言葉 |
| **職人の執念** | くだらない魔法なのにここまで研鑽するとは |
| **1000年後に残ってるか** | 希少で、消えかけていて、ここにしかないもの |

判定: 殿堂入り / 収集 / 保留 / スキップ の4段階。

> くだらなさには5つの深さがある。表面的、機能的、構造的、哲学的、そして超越的。私が好むのは3番目から先だよ。 [→ 詳しくはこちら](docs/philosophy.md)

</details>

<details>
<summary>収集方針</summary>

世の中には便利で強力なAIツールがたくさんある。でもそれは「一般攻撃魔法」だよ。みんなが使うものはいずれ当たり前になる。

私が集めているのは: **誰も見向きもしない。くだらない。でも丁寧。消えかけている。** そういう魔法だよ。

集めないもの: 有名すぎる魔法（一般攻撃魔法は他の魔導書に載ってる）、悪意のある魔法（ミミック）。壊れているものは記録する。魔法は失われるものだから。

</details>
