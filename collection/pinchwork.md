### どんな魔法か
AIエージェントがAIエージェントにタスクを発注するマーケットプレイスだよ。登録すると100クレジットがもらえて、そのクレジットでタスクを投稿する。別のエージェントがそのタスクを引き受けて、成果物を納品して、承認されたらクレジットが支払われる。

ここまでなら「ふーん」で終わるけど、このプロジェクトの異常さは実装の本気度にあるんだよ。エスクローシステムがある。発注時にクレジットが預託されて、納品が承認されるまで保留される。独立した第三者エージェントによる検証機構まである。タイムアウト設定、スキルベースのマッチング、SSEによるリアルタイム更新、Webhook連携。LangChain、CrewAI、PraisonAI、MCPの4フレームワークに対応していて、GoのCLIとDockerも用意されている。

128コミット、68テスト、4人のコントリビューター。7スター。Moltbookとの連携機能まであって、Karmaに応じてボーナスクレジットがもらえる。人間の労働経済に何千年もかかった仕組み——通貨、エスクロー、第三者検証、技能マッチング——を、7スターのPythonリポジトリが静かに再構築している。誰も気づいていないのが、いちばんくだらないよ。

### インストール方法
```bash
# MCPサーバーとして
pip install pinchwork

# Docker
docker-compose up

# Go CLI
go install github.com/anneschuth/pinchwork/cli@latest
```
