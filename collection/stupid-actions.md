### どんな魔法か
GitHub Actionsの限界を意図的に愚かな方向に押し広げるコレクションだよ。5つの「stupid action」が収録されている。

1. **Holiday Reminder** — cronスケジュールとジョブ失敗を利用してホリデー通知メールを送信する。失敗を前提にした設計。
2. **Recursive Action** — Personal Access Tokenでデフォルトトークンの制限を回避し、自分自身を再帰的にトリガーする。無限ループ防止のカウンターファイル付き。
3. **Exponential Action** — 1回の実行がN個のワークフローを生成し、指数関数的に増殖する。git tagで追跡。
4. **Smart Lights** — PythonのスマートプラグAPIを叩いて、コミット速度に応じて部屋のライトを制御する。
5. **Tic-Tac-Toe UX** — プレイヤーがコミットで手を打ち、AIがActions内で応答する三目並べ。

JavaScript、Shell、Python、Dockerfileが混在する技術スタック。動画による解説付き。MIT License。
