### どんな魔法か
gitのコミットメッセージに、AIが選んだ絵文字を付けてくれる魔法だよ。`gitemoji` とターミナルで打つと、直前のコミットの内容をGPT-3が読んで、最も適切な絵文字を1つ提案してくれる。承認すればコミットメッセージの先頭に絵文字が追加される。それだけだよ。

絵文字1個のためにOpenAI APIを叩いている。考えてみてほしい。世界のどこかのサーバーで、数十億パラメータのニューラルネットワークが電力を消費して、コミットメッセージを読んで、最も適切な絵文字を1個選んでいる。バグ修正なら虫の絵文字、ドキュメント更新なら本の絵文字。人間なら3秒で決められることを、GPT-3に委ねている。

でもね、この魔法の本当の価値は、5スターで16コミットで、CC0ライセンスだということだよ。CC0は「著作権を放棄する」という宣言だよ。つまりこの作者は、「絵文字1個を選ぶためのAI CLIツール」を作って、それを完全に自由に使えるように世に放って、何の見返りも求めていない。gitmoji（4.6kスター）やdevmoji（467スター）のような「絵文字コミット規約」は、ルールを決めてそれに従わせる。でもgit-emojiは「AIに聞けばいいよ」と言っている。ルールを覚える必要すらない。怠惰の極致を、Goの美しい簡潔さで実装している。

### インストール方法
```bash
go install github.com/ammario/git-emoji/cmd/gitemoji
export OPENAI_API_KEY="your-key"
gitemoji

# オプション: git alias
# ~/.gitconfig に追加:
# [alias]
#   emoji = "! ~/go/bin/gitemoji"
```
