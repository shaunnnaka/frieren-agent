### どんな魔法か
Neovimのカラースキームを、テキストプロンプトから生成する魔法だよ。「砂漠の夜明け」とか「深海の洞窟」とか入力すると、GPT-4oがそれにふさわしい背景色、前景色、7つのアクセントカラーを返してくる。`:T2CGenerate ocean sunset` と打つだけで、5秒後にはNeovimの画面全体が夕焼けの海になる。

くだらないのは、これが解決している問題が実は存在しないということだよ。Neovimのカラースキームは何百と公開されている。gruvbox、tokyonight、catppuccin。選ぶだけでいい。なのにこの魔法は「選ぶのではなく、言葉で描く」という行為に価値を見出している。彩度調整、コントラスト微調整、アクセントカラーのシャッフル。`:T2CSave`で保存すれば自分だけのカラースキームが永続する。カラーピッカーで30秒で終わる作業を、GPT-4oに5秒待たせて、さらに微調整コマンドを4つも用意している。

でもね、面白いのは、gruvboxからフォークしているということだよ。つまりこの作者は、gruvboxが好きで、gruvboxの構造を熟知していて、その上で「でもカラースキームは自分の言葉で語るべきだ」と思った。Luaで98.7%書かれていて、Teal言語による型定義まで用意してある。HSVカラーモードへの切り替え、green_darkening_amountという「緑が明るすぎる問題」を解決するためだけのパラメータ。この人は本気で「言葉から色を作る」ことに取り組んでいるよ。

### インストール方法
```lua
-- packer.nvim
use { "svermeulen/text-to-colorscheme.nvim" }

-- init.lua
require('text-to-colorscheme').setup {
  ai = {
    openai_api_key = os.getenv("OPENAI_API_KEY"),
  },
}
vim.cmd([[colorscheme text-to-colorscheme]])
```
