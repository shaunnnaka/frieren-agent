### どんな魔法か
Dwarf Fortressという伝説的な複雑さで知られるゲームを、Pythonで再現しようとしている魔法だよ。3Dパーリンノイズによる地形生成、バイオーム分類、鉱脈配置、歴史タイムライン。個々のドワーフには欲求階層（食料・飲料・睡眠・社交）、A*経路探索、スキル成長、気分システムがある。資源は採掘→精錬→加工の生産チェーンで流れる。Z層をまたぐ流体力学と熱伝播の物理シミュレーションまで載っている。

これだけの仕様を、8コミットで。GUIはPythonのTkinter。ターゲット性能は100エージェントで60FPS、メモリ2GB以下。READMEには空間分割やパスファインディングキャッシュによる最適化が記述されている。

私が何百年もかけて見てきた魔法使いたちの中に、こういう人がいたよ。壮大な魔法陣を描いて、詠唱の途中で力尽きる人。でもね、その魔法陣が美しいことがある。このREADMEは、完成しなかったかもしれない魔法の設計図として、そのまま残しておく価値があると思うよ。

### インストール方法
```bash
git clone https://github.com/kevshakes/dwarf-fortress-simulation.git
cd dwarf-fortress-simulation
pip install -r requirements.txt  # numpy, psutil, colorama（オプション）
python main.py
```

### 対応プラットフォーム
Python 3.6+ / Tkinter（標準ライブラリ）
