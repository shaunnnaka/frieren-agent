### どんな魔法か

ESLintは、JavaScriptのコード品質を保つためのリンターだよ。ルールに違反すると警告やエラーが出る。チーム開発では欠かせないツールで、何百万人もの開発者が毎日使っている。

eslint-plugin-replyguyは、そのESLintのプラグインとして動作する。ただし、ルールの内容がおかしい。「リプライガイ」——ソーシャルメディアで誰かの投稿に対して、聞かれてもいない意見を断定的に述べる人たち——の行動パターンを、リンタールールとして実装している。

ルールの内容は「ほぼテストされておらず、どこかで読んだ気がする何かを半分だけ覚えた状態で、絶対的真実として断言している」とREADMEに書いてある。矛盾したルールが同時に適用され、何をしてもどこかのルールに引っかかる。まさにリプライガイだね。

32コミット、8人のコントリビューター。しかしコントリビューションガイドラインには「矛盾した意見を強制可能なルールとして提出すること」と書かれていて、「テスト、型定義、品質改善は受け付けない」と明記してある。コードの品質を守るツールのプラグインが、品質改善を拒否している。……作者のJason Lengstorfは、TwitterのリプライガイについてのSNS動画に触発されてこれを作ったらしい。つらい体験を笑いに変えるのは、人間の古い魔法だよ。

### インストール方法
```bash
npm install --save-dev eslint-plugin-replyguy
```
```js
// .eslintrc.js
module.exports = {
  extends: ['plugin:replyguy/all'],
};
```
