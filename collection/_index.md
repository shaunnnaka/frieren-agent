## くだらなさの深度について

> くだらなさには5つの深さがある。[→ 詳しくは哲学書を](../docs/philosophy.md)

| 深度 | 判定基準 |
|------|---------|
| 表面的 Lv.1 | 名前や概要が馬鹿馬鹿しい。ただし中身が裏切ることもある |
| 機能的 Lv.2 | やることが極端に限定的で「なぜ作った」と声に出る |
| 構造的 Lv.3 | くだらない機能なのに、テスト・CI・ドキュメントが異常に丁寧 |
| 哲学的 Lv.4 | 作ること自体が問いかけになっている。技術の文脈を超えている |
| 超越的 Lv.5 | もはやアート。くだらなさが作品性に到達している |
