## Cat Facts MCP Server

> フリーレンの感想：「レート制限があるよ。毎秒5回、月10万回まで。……猫の豆知識を月に10万回取得する状況って、どういう状況なの。それを想定して実装している人がいるの。いるんだね」

**くだらなさの深度:** 構造的（Lv.3）— 猫の豆知識を返すだけなのに、レート制限・モジュラーアーキテクチャ・CLIオプションを備えている

**GitHub Stars:** 1（2026-02-25時点）

### どんな魔法か

catfact.ninja APIから猫に関する豆知識を取得するMCPサーバーだよ。できることは2つ。猫の豆知識を1つ返すか、複数返すか。それだけだよ。最大文字数を指定できるけど、それも1〜500文字の範囲に制限されている。猫の豆知識に500文字制限をかける必要があったのかな。

でもソースコードを見ると、驚くほど丁寧に作られているよ。CatFactsClientクラスにはレート制限の実装がある。毎秒5リクエスト、月10万リクエストまで。タイムスタンプベースのリセット機構。猫の豆知識のためのレート制限だよ。サーバークラスはエラーハンドリング、グレースフルシャットダウン、SIGINTハンドラーまで備えている。猫が急にいなくなっても安全に停止するんだね。

TypeScript 93.4%。型定義ファイル、設定管理、CLIオプション（トランスポート選択、ポート設定、ホスト指定）。環境変数は10個。CAT_FACTS_API_URL、DEFAULT_LIMIT、MAX_LIMIT、MIN_LENGTH、MAX_LENGTH。猫の豆知識の最小文字数を環境変数で管理する人を、私は見たことがなかったよ。10コミット。1スター。そのスターは自分でつけたのかもしれないね。

### インストール方法
```bash

cd cat-facts
npm install
npm run build
npm start
```
