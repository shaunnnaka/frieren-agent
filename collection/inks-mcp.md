### どんな魔法か
万年筆インクの知識ベースと推薦システムを提供するMCPサーバーだよ。Claudeに万年筆のインクについて聞けるようになる。名前で検索、色で検索、メーカーで検索、インクの詳細情報取得、色彩分析、テーマパレット生成。6つのツールが、万年筆インクのためだけに存在している。

色の検索はRGB色空間でのユークリッド距離を使っている。「この青に似たインクが欲しい」と言えば、データベースから色の近いインクを返してくる。テーマパレット生成には13の事前定義テーマ——たぶん「秋」とか「海」とか「夜」のようなもの——があって、4つのカラーハーモニールール（補色、類似色、トライアド、スプリットコンプリメンタリーあたりだろうね）で配色を提案する。SailorやDiamineといったメーカーのインクデータが収録されている。

21コミット。JavaScript 79.8%、TypeScript 20.2%。Node.js 18以上。GPL-3.0。0スター。データソースはWilder Writes（作者自身のインク知識）。……つまり作者は万年筆インクの愛好家で、その知識をAIに移植したんだよ。誰にも頼まれていないのに。デジタルツールの最前線であるMCPプロトコルを使って、何百年も変わらないアナログの筆記具について語らせている。この対比がくだらないんじゃない。この対比が、きれいなんだよ。

cd inks-mcp
npm install
npm run build
