### どんな魔法か
AIエージェントが実際にピザ屋に電話をかけて、人間の店員と会話して、ピザを注文するツールだよ。

技術スタックはこう。ユーザーのリクエストをMistral-7Bがパースして、適切なピザ屋の電話番号にルーティングする。Twilio APIで電話をかける。店員の声をVoskでローカル音声認識する。ChatGPTが文脈を理解して応答を生成する。Google Text-to-Speechが応答を音声にする。これがリアルタイムで回るんだよ。

4時間のハッカソンで作られた。68コミット。開発中に実際に6軒のピザ屋に電話をかけた。最初のバージョンはAIの応答が冗長すぎて、店員がイライラして電話を切られたらしい。Twilioのデバッグが難しくて、電話の途中で何が起きているかわからなかったとも書いている。

ピザを注文するという、人類が数十年前から電話一本でできていたことを、5つのAIモデルと通信APIを組み合わせて自動化した。しかも店員に怒られた。くだらないね。

### インストール方法
```bash
git clone https://github.com/freeman-jiang/PizzaGPT.git
cd PizzaGPT
pip install -r requirements.txt

# 必要な環境変数:
# TWILIO_ACCOUNT_SID, TWILIO_AUTH_TOKEN
# OPENAI_API_KEY
# Mistral-7B ローカルモデル
# Vosk音声認識モデル
```
