### どんな魔法か
Raspberry Piにカメラとペリスタルティックポンプを繋いで、人の顔を撮影し、表情から感情を読み取り、その感情に合ったカクテルを自動で調合する装置だよ。

パイプラインはこう。サーボ搭載カメラが顔を撮影する。AWS Rekognitionが年齢・感情を解析する。感情に基づいてレシピデータベースからカクテルを選ぶ。4チャンネル12Vリレーモジュールがフードグレードのペリスタルティックポンプ（毎秒1.9mL）を制御して、各材料を正確な比率で注ぐ。完成したらAWS Pollyが多言語で「お飲み物ができました」と案内する。

18コミット。Python 100%。GPIO制御、スレッド化されたポンプ操作、レシピ管理システム。悲しそうな顔をした人にカクテルを作るために、AWSの2サービスとRaspberry Piのハードウェア制御をフルに使い切っている。このパイプラインの各段階に、作者が「これは正しいことだ」と確信しながら書いたコードがあるんだよ。たぶん。

### インストール方法
```bash
git clone https://github.com/saubury/cocktail-pi.git
cd cocktail-pi
pip install -r requirements.txt

# ハードウェア: Raspberry Pi + カメラモジュール + 4ch リレー + ペリスタルティックポンプ
# AWS: Rekognition + Polly の認証情報が必要
```

### 対応プラットフォーム
Raspberry Pi / Python / AWS Rekognition / AWS Polly
