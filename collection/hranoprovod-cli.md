### どんな魔法か
YAMLファイルでカロリーと栄養素を記録するコマンドラインダイエットトラッカーだよ。Go言語で書かれている。

2つのYAMLファイルで全てが動く。`food.yaml` にレシピと栄養データを書く。`log.yaml` に日々の食事を記録する。レシピは入れ子にできる。たとえば「サラダ」のレシピに「ドレッシング」のレシピを含めて、栄養素が再帰的に計算される。

機能は豊富だよ。日別レポート、バランスツリー、CSVエクスポート、統計サマリー、バリデーション（lint）。Dockerでも動く。v3.0.0まで到達している。330コミット。11スター。

何がくだらないかって？ YAMLファイルを手書きしてカロリーを管理するという行為自体は別にくだらなくない。でも330コミットかけてGoで書いたCLIツールが、11スターなんだよ。MyFitnessPalやCronometerが存在する世界で。誰にも頼まれていないのに、YAMLでダイエットを管理する仕組みを10年以上磨き続けている人がいる。その静かな執念を記録に残したい。

### インストール方法
```bash
# Go install
go install github.com/aquilax/hranoprovod-cli/v3/cmd/hranoprovod-cli@latest

# Docker
docker run -it --rm -v $(pwd)/examples:/data aquilax/hranoprovod-cli -d /data/food.yaml -l /data/log.yaml

# Homebrew (非公式)
# なし
```
