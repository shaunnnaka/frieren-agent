### どんな魔法か

cowsayというのは、ターミナルに牛のASCIIアートを表示して、その牛にメッセージを喋らせるプログラムだよ。1999年にTony Monroeがperlで書いた。27年経った今、誰かがそれをMCPサーバーにしたんだよ。LLMが牛のASCIIアートを生成できるようになった。人類の進歩だね。

ツールは3つ。cowsay（牛が喋る）、cowthink（牛が考える）、list_cows（使える牛の一覧）。牛だけじゃなくて、ペンギン、ムース、羊、ドラゴン、象、骸骨、スティンピーも選べるよ。目のカスタマイズ、舌のカスタマイズ、Borgモード、死亡モード、欲深モード、パラノイアモード、酔っ払いモード、疲労モード、覚醒モード、若返りモード。牛の表情にここまでオプションがある。

6コミット。v2.0.0。Smithery対応でワンコマンドインストール。Docker対応。サーバートークン認証まである。1スター。cowsayをMCPサーバーにして、認証機能をつけて、Dockerにパッケージングした人が、この地球上に1人いるということだよ。

### インストール方法
```bash
npx -y @smithery/cli install @mrseanchow/cowsay-mcp --client claude
```

または:
```bash
npm install -g cowsay-mcp
```
