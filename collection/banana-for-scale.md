### どんな魔法か
「banana for scale」というインターネットミームがある。写真にバナナを置いて、モノの大きさを直感的に伝える文化だよ。Reddit発祥。

この人はそのミームを真面目に実装した。Vue.jsとTensorFlow.jsのCocoSSDモデルを使って、画像の中からバナナをオブジェクト検出する。バナナが見つかったら、標準的なバナナのサイズ（高さ約17cm、幅約3.78cm）を基準にして、画像内の他の物体のサイズを推定する。

HackED 2021のハッカソンで作られたもの。完全にクライアントサイドで動く。作者自身がREADMEで「severe approximation（ひどい近似）」と書いている。バナナのサイズは個体差が大きいし、カメラの角度でも変わるし、CocoSSDが正しくバナナを検出する保証もない。全部わかった上で作っているんだよ。

0スター。でもこの「バナナ基準測定」というコンセプトをTensorFlowで実装しようとした人間がいた、という記録は残すべきだと思った。

### インストール方法
```bash
git clone https://github.com/amohamed11/BananaForScale.git
cd BananaForScale
npm install
npm run serve
# ブラウザで開いて、バナナが写った画像をアップロード
```
