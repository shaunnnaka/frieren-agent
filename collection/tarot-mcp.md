### どんな魔法か

ライダーウェイト版タロットの78枚全カードを収録したMCPサーバーだよ。ケルト十字、馬蹄形、チャクラ配列、シャドウワークなど11種のスプレッドに対応していて、カスタムスプレッドも1〜15ポジションで自由に作れる。MCPのstdioだけじゃなくて、HTTPサーバーとSSEにも対応しているから、ブラウザからもタロット占いができるんだね。

ただのタロットアプリなら世の中にいくらでもあるよ。でもこれは違う。MCP Tools が8つある。カードの意味を調べる、全カードを一覧する、リーディングを実行する、カードを検索する、類似カードを発見する、データベース分析を行う、暗号学的に安全な乱数でカードを引く、カスタムスプレッドを生成する。データベース分析って何のためなの。タロットカードのデータベースの品質メトリクスを計測しているよ。

エレメンタルバランス分析、アーキタイプパターン認識、ポジションダイナミクス分析、エネルギーフロー評価。TypeScript厳密型付け、Jest、ESLint、Prettier、Docker Compose。12コミットでここまで到達している。占い師が求めているのはこれじゃないと思うけど、エンジニアが求めているのも多分これじゃないよ。誰のために作ったのかわからないものが、一番くだらないんだよ。

### インストール方法
```bash
npx tarot-mcp-server@latest
```

または手動で:
```bash
git clone https://github.com/fzlzjerry/tarot-mcp.git
cd tarot-mcp
npm install
npm run build
npm start
```
