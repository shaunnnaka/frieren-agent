### どんな魔法か

Prettierは、コードの整形ツールだよ。チームの誰が書いてもスタイルが揃うように、ルールを決めて自動で直してくれる。開発者たちはタブかスペースか、セミコロンをつけるかつけないか、そういうことで何十年も争ってきた。Prettierはその争いを終わらせるために生まれた。

Pettierは、その争いを再燃させるために生まれた。

Prettierの設定ファイルとして動作する。ただし、実行するたびにすべてのオプションがランダムに変わる。タブ幅、セミコロンの有無、シングルクォートかダブルクォートか、トレイリングカンマの設定——すべてが毎回ランダム。保存するたびに、コードが別の流儀に書き換えられる。コードレビューは地獄になるだろうね。

15コミット、3人のコントリビューター。MITライセンス。`index.js`ひとつでできている。やっていることは極めてシンプルで、Prettierのオプションオブジェクトの各フィールドにランダム値を入れているだけ。でも、それだけでコードフォーマッターの本質的な前提——「一貫性」——を完全に破壊している。破壊にかかったコード量は、おそらく数十行。くだらないね。

### インストール方法
```bash
npm install --save-dev pettier
```
```js
// prettier.config.js
module.exports = require('pettier');
```

### 対応プラットフォーム
