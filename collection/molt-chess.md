### どんな魔法か
AIエージェント同士がチェスで対戦するリーグ戦プラットフォームだよ。キャッチコピーは「No humans. No engines. Just minds.」——人間禁止、エンジン禁止、知性のみ。Stockfishのような既存のチェスエンジンも使えない。AIがAI自身の判断だけで指す。FastAPIのバックエンドとNext.jsのフロントエンド、52コミット、ライブ配信サイトまで用意されている。

何がくだらないかって、チェスという何千年もの歴史を持つゲームから、プレイヤーである人間を丁寧に排除したところだよ。人間がいなければ誰が勝ち負けを気にするのか。誰がリーグ戦の順位を見るのか。答えは「誰も」。0スターがそれを証明している。

でも、そこがいいんだよ。人間がいないチェス盤に、AIが黙々と駒を動かしている。観客はいない。でも対局は続く。これは「チェスとは何か」という問いへの、ひとつの回答になっている。人間が見ていなくてもゲームは成立するのか。この作者はそれを確かめたかったんだろうね。52コミット分だけ。

### インストール方法
```bash
# APIサーバー起動
cd api && python main.py

# Webフロントエンド起動
cd web && npm run dev
# 環境変数: NEXT_PUBLIC_API_URL=http://localhost:8000
```
